Server API
==========
Version: 1.4.1

Container Classes
-----------------
Player
    id     : int
    teamId : int
    coord  : Coordinates

Food
    id    : int
    team  : int
    coord : Coordinates

Points
    total   : int
    earners : HashMap<Player, int>

Coordinates
    x : int
    y : int

MessagePlayerId
    id     : int
    teamId : int
    coord  : Coordinates

MessagePlayerCoord
    player    : MessagePlayerId
    newCoord : Coordinates

MessageFoodCoord
    food      : Food
    newCoord : Coordinates
    gone      : boolean

MessagePlayerName
    playerName : String
    teamId     : int

MessagePointsUpdate
    player   : MessagePlayerId
    newPoint : int

MessageWorldParams
    worldSizeX  : int
    worldSizeY  : int
    gridSize    : int
    playerSpeed : int


Events FROM server
------------------
* connect
* disconnect
    - MessagePlayerId
* gameStart
* gameEnd

# The information about the world.
* worldParams
    - MessageWorldParams

# The identity of the player. This includes the starting ID, team ID, and the
# starting coordinates of the player who received this event.
* playerId
    - MessagePlayerId

# Whenever somebody (including yourself) moved. The new coordinate of that
# user is stored in MessagePlayerCoord.newCoord
* playerUpdate
    - MessagePlayerCoord

# Whenever there's a change in food coordinate, that information is sent from
# the server. When `gone' is true, somebody took that food. If false, then
# there is a new food that appeared on the field at location
# MessageFoodCoord.newCoord
* foodUpdate
    - MessageFoodCoord

# The updated points of the team
* pointsUpdate
    - MessagePointsUpdate


Events TO server
----------------
* connect
    - MessagePlayerName
* disconnect
* move
    - MessagePlayerCoord

Optional
* teamSelected
